#### Trie树

即字典树，又称单词查找树或键树。利用字符串的公共前缀来降低查询时间的开销以达到提高效率的目的。典型应用是用于统计和排序大量的字符串（但不仅限于字符串），所以经常被搜索引擎系统用于文本词频统计。优点是：最大限度地减少无谓的字符串比较，查询效率
比哈希表高。

#### 并查集

适用于组团、配对问题的一种特殊结构，

#### AVL树

由二叉搜索树延伸而来，如何保证二叉树的搜索效率，就需要考虑二叉树的平衡性。AVL是一种平衡的二叉搜索树（平衡因子差值不超过2）。通过4种旋转来保证平衡因子在{-1,0,1}区间内。

#### 红黑树

一种近似平衡的二叉搜索树，确保任何一个结点的左右子树的高度差小于两倍。
